===============================================================================
INSTRUKTIONER TIL GENERERING AF LYDFILER
===============================================================================

1. Kopier script teksten
2. Gå til TTS generator (fx elevenlabs.io, Google Cloud TTS, Amazon Polly)
3. Vælg dansk stemme
4. Generer MP3
5. Gem med det angivne filnavn
6. Upload til sounds/ mappen på din webhosting

ALTERNATIV - BROWSER TTS:
Du kan også bruge browserens indbyggede TTS via JavaScript:
- speechSynthesis.speak(new SpeechSynthesisUtterance("tekst her"))
- Dette kræver ingen eksterne filer, men kvaliteten varierer

ANBEFALEDE TTS TJENESTER:
- ElevenLabs (bedste kvalitet, dansk tilgængelig)
- Google Cloud Text-to-Speech
- Amazon Polly
- Microsoft Azure Speech
